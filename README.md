# SmartContent-Automator (AI-Powered)

Un outil d'automatisation en Python qui utilise l'IA (**Gemini 2.5** via l'API officielle Google GenAI) pour transformer instantanément n'importe quel contenu textuel ou article en assets de communication : un post LinkedIn optimisé et un script vidéo au format vertical (9:16) pour TikTok/Shorts.

##  Technologies & Compétences démontrées
* **Langage :** Python 3
* **Intelligence Artificielle :** SDK Google GenAI (`gemini-2.5-flash`)
* **Concepts Clés :** Gestion des variables d'environnement (`python-dotenv`), Architecture modulaire, Gestion rigoureuse des exceptions (API, Connexion, IO).
